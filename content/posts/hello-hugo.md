## 新建文章

```
hugo new posts/my-first-post.md
```
或者直接在 `content/posts`目录手动创建`my-first-post.md`

## Start the Hugo server

```
hugo server -D
```
启动后可以本地预览


## Build static pages

```
hugo -D
```
